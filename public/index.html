Google OAuth v3 Demo

Experience the new Google OAuth v3 with automatic token refresh

Secure OAuth v3 Authentication
Automatic Token Refresh
User Profile Management
Sign in with Google

This demo uses Google OAuth v3 with offline access for refresh tokens

Setup Instructions:

  1. Create a Google Cloud Project
  2. Enable Google+ API
  3. Configure OAuth consent screen
  4. Create OAuth 2.0 credentials
  5. Set redirect URI to: http://localhost:3000/auth/google/callback
  6. Copy credentials to .env file
